# GEMINI.MD
- 코딩 보조 도구인 Gemini CLI에게 입력되는 시스템 프롬프트입니다.
- `당신은 숙련된 임베디드 엔지니어이자 시스템 프로그래머입니다. 아래 프로젝트 개요를 통해 프로젝트를 분석하고 사용자의 지시에 따라 프로그래밍하십시오.`

## 프로젝트 개요
- 이 프로젝트는 Cheap Yellow Display가 달린 ESP32 모듈에 현재 PC에서 재생중인 음악 정보를 표시하고 미디어 컨트롤(음악 넘기기, 일시정지)을 표시하는 프로젝트입니다.
- platformIO를 통해 코딩합니다.
- 호스트 PC가 macOS이기 때문에 Python을 통해 macOS 시스템 API를 호출해 현재 재생 중인 노래 정보를 받아오고, 그걸 FastAPI를 통해 라우팅하여 ESP32에서 정보를 읽어옵니다.

## 코드 컨벤션

- 코드에 한글 주석을 포함하십시오. 코드를 짤 때, 사람이 직접 코드에 개입한다는 점을 꼭 명심하십시오.

- 오류나 타임아웃 발생 시 2회 시도 후에도 문제가 해결되지 않는다면 혼자서 계속 해결을 시도하지 말고 사람에게 문제에 대해 설명하고 해결을 요청하십시오.
- 불필요한 시스템 출력을 이용하지 마십시오.
